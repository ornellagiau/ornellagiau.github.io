Quadra

2013

Quadra is a visual search tool for digital libraries archives.

Â 

The project investigated how and if is possible to engage users more in their search activities and which techniques are better suited to represent data like bibliographical networks, citation patterns and literary text.

Also, standing on these considerations, I developed a digital interface for scientific papers database exploration, targeting students and researchers as support for their daily work tasks.

Role
Concept, UX/UI Design, Prototype
